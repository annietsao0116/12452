12.4
八面體取代的實驗證據
Experimental Evidence in Octahedral Substitution

八面體錯合物的取代反應，依其機制可分為 解離（D / Id）、 締合（A / Ia），以及 交換（I）型。 本章節重點在於整理各種實驗方法，以判斷真實反應究竟走哪條路徑。

整個 12.4 章節的核心問題是：
「我們怎麼從實驗數據判斷一個反應是 D/Id 還是 A/Ia 機制？」

手寫筆記提醒：判斷機制需要多項證據綜合，不能靠單一實驗。
簡報中五個支持 D 機制的證據、以及 LFER、締合機制的特徵，都要記熟。
12.4.1
解離機制
Dissociation — Ligand Field Effect & Electronic/Steric Factors

在解離機制中，八面體錯合物先失去一個配基 X，形成五配位過渡態（方錐形 SP 或三角雙錐形 TBP），再由新配基 Y 填入空位。

⚡ 配位場活化能 LFAE（Ligand Field Activation Energy）

LFAE = 方錐形（SP）過渡態的 LFSE − 八面體反應物的 LFSE

• LFAE 正值越大 → 活化能高 → 解離越困難（inert）
• LFAE 負值 → 活化能低 → 解離容易（labile）

🔬 電子效應與空間效應（Electronic & Steric Factors）

以下兩個規律均源自「金屬與配基間靜電吸引力越大，解離越慢」：

1. 中心離子氧化態↑ → 解離速率↓（更難離去）
 例：[AlF₆]³⁻ > [SiF₆]²⁻ > [PF₆]⁻ > SF₆（3+→4+→5+→6+，速率依次遞減）

2. 離子半徑↑（半徑越小越慢）
 例：[Sr(H₂O)₆]²⁺ > [Ca(H₂O)₆]²⁺ > [Mg(H₂O)₆]²⁺（112pm→99pm→66pm）

例外：第一列過渡金屬的 [M(H₂O)₆]²⁺ 雖然半徑小，但半衰期均 <1 秒（labile），
這是因為電子組態（LFAE 負值）主導，而非離子半徑。

以下是支持解離機制（D / Id）的五大實驗證據（點擊展開詳細說明）：

  • 1
    速率常數對不同 Y 變化不大（<10倍）
    若解離是速率決定步驟（rds），則進入配基 Y 的種類不應影響速率。
    實驗：[Co(NH₃)₅(H₂O)]³⁺ 與各種陰離子反應，k₁ 值相差不超過一個數量級（N₃⁻: 100, SO₄²⁻: 24, Cl⁻: 21, NCS⁻: 16 × 10⁻⁶ s⁻¹）。
    → 符合 D 機制特徵。
    ▶
  • 2
    錯合物電荷↑ → 取代速率↓
    反應物電荷越正，金屬-配基靜電吸引力越強，配基越難解離。
    → 支持「M-L 鍵斷裂是速率決定步」（D 機制邏輯）。
    ▶
  • 3
    空間擁擠（steric crowding）↑ → 解離速率↑
    配基擁擠時，某一配基被「推出去」更容易，有利於解離。
    注意對比：若為 A 或 Ia 機制，空間擁擠反而會阻礙新配基進入，速率↓。
    → 速率↑ 支持 D，速率↓ 支持 A。
    ▶
  • 4
    速率與 M-L 鍵強度呈線性自由能關係（LFER）
    離去基（X）與金屬的鍵越強，解離越慢，log k 與 log K 呈線性關係。
    → 熱力學參數（鍵強度）主導動力學（速率常數），符合 D 機制（詳見 12.4.2）。
    ▶
  • 5
    活化體積 ΔV‡ 為正值
    D 機制：一個物種分裂為兩個 → 體積膨脹 → ΔV‡ > 0
    A 機制：兩個物種合併為一個 → 體積縮小 → ΔV‡ < 0
    → 可通過改變壓力測量速率常數的變化來判斷。
    注意：高電荷離子的溶劑化效應可能干擾結果，需謹慎詮釋。
    ▶

12.4.2
線性自由能關係（LFER）
Linear Free-Energy Relationships

當M-L 鍵強度（熱力學參數）是決定解離速率的主要因素時， 動力學與熱力學之間會出現線性相關性。

反應：[ML₅X]ⁿ⁺ + Y ⇌ [ML₅Y]ᵐ⁺ + X log k（速率常數） vs. log K（平衡常數）呈線性 → LFER 成立 數學依據： ln k = ln A − Eₐ/RT （Arrhenius 方程） ln K = ΔS°/R − ΔH°/RT （van't Hoff 方程） 若一系列反應的 A 和 ΔS° 相近，且 Eₐ ∝ ΔH°，則 ln k ∝ ln K
LFER 的直覺理解：
X 離去基與金屬的鍵越強（K 越小，熱力學上越難被取代），
解離也越慢（k 越小），因此 log k 和 log K 線性正相關。

重要！ LFER 是間接證明反應機制的工具——
若 log-log 圖是直線，說明熱力學鍵強度主導了動力學速率，
這正是解離機制的核心特徵（M-X 鍵斷裂是 rds）。

📋 [Ni(H₂O)₆]²⁺ 的取代反應：Pre-association 機制

對 [Ni(H₂O)₆]²⁺ 而言，k₂ 值對不同配基 Y 差異不大（在 5 倍以內）， 且接近水交換的速率常數。
但 K₁（離子對形成常數）差異顯著 — 反應速率主要受離子對形成（K₁）影響。
→ 符合 pre-association（預締合）機制（也稱 Id 機制的一種）

12.4.3
締合機制
Associative Mechanisms — less common in octahedral complexes

締合反應在八面體錯合物中較不常見（因為八面體中心已有六個配基，空間限制大）。 但在某些特殊情況下，仍可觀察到 Ia 或 A 的特徵。

📤 解離型 Id（[Cr(NH₃)₅(H₂O)]³⁺）
  • 不同 Y 的速率常數相差 <6 倍
  • 進入配基影響小
  • ΔS‡ 近零或正值
  • NH₃ 提供較多電子密度→ Cr 中心較富電子→ 對新親核試劑不敏感
📥 締合型 Ia（[Cr(H₂O)₆]³⁺）
  • 不同 Y 的速率常數相差 >2000 倍
  • 進入配基影響大
  • ΔS‡ 為負值（有序化的過渡態）
  • H₂O 提供電子密度少→ Cr 中心缺電子→ 對親核試劑敏感
關鍵概念：e⁻ 密度決定機制！
NH₃ 的 e⁻ 捐贈能力 > H₂O → [Cr(NH₃)₅(H₂O)]³⁺ 中 Cr 電子密度較高
→ 對新配基的靜電吸引力降低 → 傾向解離型 Id

相比之下，[Cr(H₂O)₆]³⁺ 中 Cr 的電子密度低 → 對親核配基敏感 → 締合型 Ia

Ru(II) vs Ru(III) 也是同理：
Ru(III) → Ia（△S‡ 負，速率隨 Y 大幅變化）
Ru(II) → Id（速率對 Y 不敏感），可能因負電荷增加→ Ru-OH₂ 鍵減弱→ 傾向解離

12.4.4
共軛鹼機制 SN₁CB
The Conjugate Base Mechanism

某些顯示二階動力學（second-order kinetics）的反應，表面上像締合機制（OH⁻ 參與）， 實際上卻是單分子解離（D 機制）的本質，但需要先去質子化形成共軛鹼。

⚠️ SN₁CB 的迷人陷阱

看到「速率 ∝ [OH⁻]」時，不要立即以為是 A 機制（OH⁻ 直接進攻）！
SN₁CB 的 OH⁻ 扮演的是鹼催化去質子化的角色，不是親核試劑。
真正的 rds 是 X⁻ 解離（單分子）。

SN₁CB 反應步驟（以 [Co(NH₃)₅X]²⁺ 為例）

1
去質子化平衡：OH⁻ 從 NH₃ 配基上奪走 H⁺，形成 amido 中間體
[Co(NH₃)₅X]²⁺ + OH⁻ ⇌ [Co(NH₃)₄(NH₂)X]⁺ + H₂O
fast equilibrium
2
解離（rds）：共軛鹼（CB = amido 錯合物）失去 X⁻ → 5 配位中間體
[Co(NH₃)₄(NH₂)X]⁺ → [Co(NH₃)₄(NH₂)]²⁺ + X⁻
通常是 X 的 trans 位置上的氮被去質子化，有利於此步驟
slow (rds)
3
快速加水：H₂O 填入空位，恢復六配位
[Co(NH₃)₄(NH₂)]²⁺ + H₂O → [Co(NH₃)₅(OH)]²⁺
fast
整體：[Co(NH₃)₅X]²⁺ + OH⁻ → [Co(NH₃)₅(OH)]²⁺ + X⁻ 速率方程推導： K_h = Ka/Kw = [Co(NH₃)₄(NH₂)X⁺] / ([Co(NH₃)₅X]²⁺ [OH⁻]) rate = k · [Co(NH₃)₄(NH₂)X⁺] = k · (Ka/Kw) · [Co(NH₃)₅X]²⁺ · [OH⁻] ∴ 二階動力學，但本質是 D 機制！
為什麼 Co(III) 特別容易走 SN₁CB？
→ Low-spin Co(III) 半徑小，Lewis 酸性強，可以通過 π 相互作用穩定五配位中間體！

Trans 位效應：被去質子化的 N 通常在 X 的 trans 位，
因為 amido 基團作為 σ/π donor，可以穩定 Co(III) 的過渡態。
實驗：[Co(tren)(NH₃)Cl]²⁺ 中，離去基 trans 到去質子化 N 的異構物比 cis 的快 10⁴ 倍。

SN₁CB 的四大實驗證據（點擊展開）：

  • 1
    胺基氫的鹼催化交換（Base-catalyzed H exchange）
    直接證明去質子化平衡（步驟一）存在。NH 氫可以被鹼催化交換， 證明 amido 中間體確實會形成。
    ▶
  • 2
    ¹⁸O 同位素標記
    在 ¹⁸O 富集水中反應，產物的 ¹⁸O/¹⁶O 比例 = 溶劑的比例，與離去基種類無關。
    → 若是 A 機制，水在 rds 中進入，¹⁸O 比例應高於溶劑。
    → 實驗結果說明水在 rds 之後才進入 → 排除締合機制。
    ▶
  • 3
    RNH₂ 比 NH₃ 複合物反應更快
    空間擁擠（R 基）反而加速反應。
    → 若是締合機制，空間擁擠應該阻礙 Y 進入，速率↓。
    → 速率↑ 說明解離機制（空間擁擠有助於配基被推出去）。
    ▶
  • 4
    LFER：ln k_OH 對 ln K_OH 呈線性
    離去基鍵強度決定速率 → 確認解離（M-X 鍵斷裂）為 rds。
    ▶
12.4.5
動力學螯合效應
The Kinetic Chelate Effect

螯合配基（多牙配基）的取代反應通常比類似的單牙配基慢。 這稱為動力學螯合效應（kinetic chelate effect）。

🔗 為什麼螯合配基難以被取代？

原因一：移除螯合配基第一個鍵結點的 ΔH 比單牙配基更大（多牙配基整體結合更穩）

原因二（更重要！）：即使第一個鍵結點斷開了，
剩餘部分的鏈段仍然把「懸空」的配位原子固定在金屬附近，
使得「重新鍵結」（reattachment）的動力學障壁非常低，
比單牙配基更容易重新連上去。

→ 淨效果：整體解離（aquation）速率大幅降低

乙二胺（en）螯合配基取代的四步驟（aquation）

1
第一個 N 解離（慢）→ 懸空臂，但靠近金屬
slow
2
H₂O 快速進入空位（快）
fast
3
第二個 N 解離（慢）→ en 完全離去
slow
4
第二個 H₂O 進入（快）
fast

⚠️ 步驟1的逆反應（reattachment）也快 → aquation 整體速率↓

記憶口訣：「螯合配基難解離，因為一隻腳斷了，另一隻還抓著金屬，隨時可以重新抓上去！」

注意區分熱力學螯合效應 vs. 動力學螯合效應：
→ 熱力學：多牙配基取代後更穩定（較大的 K）
→ 動力學：解離速率很慢（不是因為沒有驅動力，而是因為離去後容易再接回來）
12.5
反應的立體化學
Stereochemistry of Reactions

解離機制下，產物的立體化學可能與反應物相同或不同， 取決於生成的五配位中間體形狀（SP 或 TBP）以及配基的進入位置。

📌 關鍵觀察（Table 12.9）

對 cis-[Co(en)₂LX]ⁿ⁺：酸性水解 → 100% cis 產物（方錐形 SP 過渡態）
對 trans-[Co(en)₂LX]ⁿ⁺：酸性水解 → cis 和 trans 混合（比例視 L 和 X 而定）

重要規律：
• cis 反應物 → 主要得到 cis 產物（retention of configuration）
• trans 反應物 → 得到 cis 和 trans 的混合物

這說明：cis 複合物更傾向形成方錐形中間體（SP）（保持構型），
trans 複合物則有更多可能走三角雙錐形（TBP）路徑（導致異構化）。
12.5.1
trans 複合物的取代
Substitution in trans Complexes

trans-[M(LL)₂BX]（LL = 雙牙配基）的解離取代， 可以沿三條解離路徑進行，生成不同立體化學產物。

路徑 (a)
方錐形中間體（SP）
B 在軸向 → X 離去 → SP 中間體
Y 從空位進入（正上方）
→ trans 產物（保持構型）
路徑 (b)
TBP 中間體（B 在赤道面）
B 在赤道面（常見）→ 三個面可進攻
→ 2/3 cis（外消旋）+ 1/3 trans
路徑 (c)
TBP 中間體（B 在軸向）
B 在軸向（少見，需要大幅重排）
→ cis（外消旋混合物）

路徑詳情

三條路徑的中間體與產物示意

路徑(b)的統計分析：TBP 中 B 在赤道面時，三個三角面都可被 Y 進攻。
→ 1/3 面讓 Y 佔與 B 反式位置 = trans 產物
→ 2/3 面讓 Y 佔與 B 順式位置 = cis 產物（理想情況，實際比例隨 L 和 X 而異）

路徑(c)為何少見？
B 要從赤道面（Oh 中的赤道）移到 TBP 的軸向位，需要：
→ 一個 N 移動 90°，另兩個 N 各移動 30°（vs 路徑 b 只需兩個 N 各移 30°）
→ LL 環在赤道面要被拉長得更多
→ 能量代價高，所以罕見
中間體類型 B 位置 主要產物 概率
方錐形（SP） B 在軸向 trans（保型） 主要路徑
三角雙錐（TBP） B 在赤道面（常見） 2/3 cis + 1/3 trans 常見
三角雙錐（TBP） B 在軸向（少見） cis（外消旋） 少見（LL 環阻礙）
12.5.2
cis 複合物的取代
Substitution in cis Complexes

cis-[M(LL)₂BX] 的解離取代也有三種中間體， 但實驗上酸性水解幾乎得 100% cis 產物，提示主要走方錐形（SP）路徑。

B 在赤道面，X 在軸向解離 → 方錐形中間體
Y 從原本 X 的位置（空位）填入
→ 保持 cis 構型（retention of configuration）

這是酸性水解（acid aquation）的主要路徑，因為得到 100% cis 產物。

B 在赤道面，X 解離後形成 TBP
三角面有三個可進攻位置：
• 2 個位置 → cis 產物（保持構型）
• 1 個位置 → trans 產物

統計上得 2/3 cis（保型）+ 1/3 trans。
鹼性取代（base substitution）中此路徑更顯著，得約 97-66% cis，約 2:1 的 Δ 構型保留。

B 在軸向，X 解離後形成 TBP（少見）
兩個三角面可進攻，第三面被 LL 環遮擋。
兩個位置均給出 cis 產物，但構型不同（Λ 和 Δ 各 50%）
→ 外消旋 cis 混合物

注意：cis 和 trans 反應物的此路徑（axial B）中間體是相同的（見 Fig 12.9c 和 12.10c 完全相同）。

General Rule（手寫筆記上的重點）：
• cis 起始物 → 主要得到 cis 產物（percentage of cis 較高）
• trans 起始物 → 往往得到 cis/trans 的平衡混合物

酸性水解 cis 複合物得 100% cis → 提示方錐形（SP）transition state
鹼性取代（SN₁CB）cis 複合物得 66-97% cis，但有部分外消旋化 → TBP 路徑也參與

  • D 機制五大證據：速率不隨 Y 改變、電荷↑速率↓、空間擁擠↑速率↑、LFER、ΔV‡>0
  • LFER：log k 對 log K 呈線性 = 離去基鍵強度主導速率 = 支持 D 機制
  • Cr(III) + NH₃配基 → Id；Cr(III) + H₂O配基 → Ia（電子密度決定機制）
  • SN₁CB：外表像 A（二階動力學），本質是 D；OH⁻ 只是催化去質子化，rds 是 X⁻ 解離
  • 動力學螯合效應：多牙配基解離慢，因第一鍵斷後容易重新連回（reattachment 快）
  • 立體化學：SP 中間體→保型；TBP（B eq）→ 2/3 cis + 1/3 trans；TBP（B ax，少見）→ 外消旋 cis
  • 通則：cis 起始物多得 cis 產物；trans 起始物常得 cis/trans 混合
